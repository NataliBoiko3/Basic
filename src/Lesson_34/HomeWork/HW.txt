 Управление заказами

Описание задачи:

Создайте программу для управления списком заказов.
Каждый объект "Заказ" должен содержать идентификатор заказа,
имя клиента и сумму заказа.

Программа должна предоставлять функциональность для добавления заказов,
удаления заказов, поиска заказов по имени клиента,
и сортировки заказов по сумме и имени клиента.

Требования:

    Создайте класс Order с полями orderID, customerName и amount.
    Используйте коллекцию для хранения объектов Order.
    Реализуйте методы для добавления, удаления, поиска и сортировки заказов.
    Реализуйте сортировку заказов по сумме и имени клиента.
    Выведите результаты выполнения каждого действия на экран.
